\section{書き込み方法}
(1)~~コントローラに接続されている24V電源以外の物を全て外す.\\
*全て接続したまま書き込みも可能.

(2)~~ISEをインストールしたPCとコントローラをJTAGで接続する.

(3)~~コントローラの24V電源を入れる.

(4)~~ISEを起動し,File$\rightarrow$Open Projectからプログラムを開く.

\begin{figure}[H]
	\centering
	\includegraphics[width=110mm]{gaze/app_E/4}
	\caption{ISE File画面}
%	\label{fig:　}
\end{figure}

(5)~~左側タブ内の以下の項目について緑色のチェックマーク(または注意△！マーク)になっていることを確認する.\\
Synthesize –XST\\
Check Syntax\\
Implement Design\\
Generate Programming File\\
もしも,チェックされていない場合は,それぞれ右クリックし,Runをクリックする.\\

(9)~~Create PROM Fileをダブルクリックする.

(10)~~PROM File Formatterが開く.以下の通りに選択する.\\
STEP1：Xilinx Flash/PROM\\
STEP2：PROM Family	→	Platform Flash\\
Dvice(bits)	→	xcf32p[32M]\\
ここでAdd Storage Deviceをクリックする.\\
STEP3：output File name→任意のファイル名(日本語が入らないようにすること)\\
output File Location→任意のフォルダを指定(日本語が入らないようにすること)\\
ここまで入力したら,OKをクリックする.\\
＊ここで作成される.mcsファイルのファイル名は,のちに選択するため覚えておく.\\
＊ファイル名は,日付と時間でつけると良い.(例：20200000$\_$0000)\\

(25)~~コントローラボードのLED6とLED8が点灯したら書き込み成功となる.\\
＊点灯していなかった場合,書き込みが出来ていない可能性があるので,もう一度やってみるか,プログラムを見直す(仕様メモリを減らす,行数をなるべく減らす等する)必要がある.

(26)~~24V電源を切り,JTAGを外す.\\
*DSPの接続をしたまま書き込みをしているなら,そのまま実機検証できる.外す必要はない.\\
基本的に,コントローラ上の配線を抜き差しする場合は,電源を切ること.
